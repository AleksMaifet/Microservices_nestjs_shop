# MicroServers

# Getting Started

• Step 1: git clone

• Step 2: prerequisites:

- Node.js

- npm install npm@latest -g

• Step 3: set your envs as an example:

- AMQP_EXCHANGE

- AMQP_LOGIN

- AMQP_PASSWORD

- AMQP_HOST

- AMQP_QUEUE

- JWT_SECRET

- MONGO_URI

- MONGO_USERNAME

- MONGO_PASSWORD

• Step 4: docker-compose up --build -V
